What is the web?

The web, or the World Wide Web (WWW), is a client server application with web pages filled with information and services. The web primarily uses the Hyper Transfer Protocol (HTTP) to access these web pages in the first place and HyperText Markup Language (HTML) to format them.

Vocabulary

The Internet: A world-wide network of networks and computers based on the Internet protocol

World Wide Web(WWW): A client server application for information and services.

HyperText Markup Language(HTML): A language that is used to create and publish static web pages and web applications.

Hyper Transfer Protocol(HTTP): A standard protocol that accesses web pages through the Internet.
